# Windows 6.1 系统补丁：KB2661332 (x64)

## 资源文件介绍

本仓库提供了一个重要的系统补丁文件：`Windows6.1-KB2661332-x64.msu`。该补丁主要用于修复Windows 6.1（即Windows 7 和 Windows Server 2008 R2）系统中的远程桌面连接蓝屏死机问题。

## 补丁详情

- **补丁编号**: KB2661332
- **适用系统**: Windows 6.1 (Windows 7 和 Windows Server 2008 R2)
- **适用架构**: x64
- **补丁类型**: 系统补丁
- **主要功能**: 修复远程桌面连接导致的蓝屏死机问题

## 补丁说明

该补丁由微软官方发布，旨在解决Windows 6.1系统在远程桌面连接时可能出现的蓝屏死机问题。此问题主要影响Windows Server 2008 R2服务器，安装此补丁后可以有效避免此类问题的发生。

## 安装方法

1. 下载本仓库中的补丁文件：`Windows6.1-KB2661332-x64.msu`。
2. 双击下载的文件，按照提示完成补丁的安装。
3. 安装完成后，建议重启计算机以确保补丁生效。

## 注意事项

- 请确保在安装补丁前备份重要数据，以防意外情况发生。
- 该补丁仅适用于Windows 6.1 (Windows 7 和 Windows Server 2008 R2) x64架构的系统。
- 如果系统已经安装了其他相关补丁，建议先卸载这些补丁后再安装此补丁。
